# Face Recognition Attendance System

An automated attendance management system powered by computer vision. This project uses Python and OpenCV to detect and recognize faces from a live video feed, automatically logging attendance with precise timestamps to eliminate manual entry and reduce proxy attendance.

##  Features
* **Real-time Face Detection:** Utilizes Haar Cascade classifiers for quick and accurate face tracking.
* **Automated Logging:** Saves attendance records instantly into a structured CSV format with precise timestamps.
* **User-Friendly GUI:** Simple desktop interface for managing student registrations and viewing attendance logs.

##  Tech Stack
* **Language:** Python
* **Libraries:** OpenCV, Tkinter, Pillow, NumPy
* **Storage:** CSV Database / File Logging
